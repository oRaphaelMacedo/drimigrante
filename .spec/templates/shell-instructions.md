# Shell Instructions

Use these shell-specific invocation rules when running CX Spec Kit scripts.

- macOS/Linux shell: use `.spec/scripts/bash/...`
- Windows PowerShell: use `.\.spec\scripts\powershell\...`
- Windows bash/Git Bash: use `powershell.exe -NoLogo -NoProfile -NonInteractive -File ./.spec/scripts/powershell/...`
- If a step shows only one shell form, use the equivalent script in the other shell.
